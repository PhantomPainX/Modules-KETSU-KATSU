What is KETSU? | ¿Qué es KETSU?

KETSU is a modular app that allows the user to create and load modules for it, allowing the user to consume any kind of media imported by the user.

Es la primera aplicación modular que permite a sus usuarios crear y cargar módulos para consumir cualquier tipo de contenido en internet, ya sean videos, imagenes e incluso recetas de cocina :)

Ketsu est une application modulaire qui permet à l'utilisateur de créer et d'importer des modules, permettant ainsi à l'utilisateur de consommer tout type de média importé par l'utilisateur.

Ketsu è un'app modulare che consente all'utente di creare e caricare moduli per essa, consentendo all'utente di consumare qualsiasi tipo di supporto importato dall'utente.

Ketsuは、ユーザーがロードしたあらゆる種類のメディアを用いられる可能にすることでモジュールを作り、載せ込むことができるのモジュラーアプリです

You can find the official page by this link: https://ketsu.app/

Documentation: https://appketsu.github.io/KETSU-Documentation/
